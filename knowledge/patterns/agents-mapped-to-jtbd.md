# Agents mapped 1:1 to JTBD with named human checkpoints

## Convergence
Five operators converge on the same agent-design pattern: don't build an "AI-augmented" abstract layer. Enumerate the jobs-to-be-done. Map each to a named agent (or agent class). Insert named human checkpoints at the alignment-critical moments. The mapping is concrete, not generic.

## Operators
- Evan Spiegel (Snap), `ins_jtbd-as-agent-wiring-diagram`. Map agents 1:1 to enumerated JTBD, not abstractly to AI-augmented workflows.
- Amol Avasare (Ramp), `ins_cash-four-stage-growth-automation`. CASH = Collect/Automate/Scale/Hire, agents handle the inner stages, humans alignment.
- Aishwarya Naresh Reganti, `ins_cccd-continuous-calibration`. Continuous Calibration, Continuous Development is the operating loop; humans calibrate, agents develop.
- Yamini Rangan (HubSpot), `ins_agent-first-gtm-flywheel` and `ins_start-with-support-fastest-ai-roi`. Three-stage flywheel with one named agent per job; start with customer support for fastest ROI.
- Kieran Flanagan (HubSpot), `ins_rebuild-gtm-around-ai`. Build a cross-functional pod that owns "AI in GTM" as one mandate, not as a bolt-on.
- Claire Vo, `ins_progressive-trust-onboarding`. Onboard agents like an EA: progressive trust, named tiers.
- Hamza Farooq, `ins_three-class-agent-taxonomy`. Tag each loop as one of three agent classes or under-resource it.

## Variation
- Spiegel and Yamini provide the *mapping discipline* (1:1 to JTBD, named agent per job).
- Avasare and Reganti provide the *operating loop* (CASH, CCCD).
- Kieran provides the *org structure* (cross-functional pod with single mandate).
- Vo provides the *trust onboarding ramp*.
- Farooq provides the *taxonomy for resourcing*.
- Convergence: refuse abstraction. Each agent has a name, a job, a checkpoint, a class.

## Implication
Before building any agent, write the JTBD list. Before deploying, name the human checkpoint and the trust tier. Before scaling, classify the agent (Spiegel/Farooq taxonomy) and decide its loop (CASH/CCCD). If you can't draw the wiring diagram on a whiteboard, you're not ready to build.
