# Status-quo framing — Q3 homepage rebuild

The pitch must establish that the buyer's existing situation is the real competitor. Loss aversion makes the felt switching cost roughly 2× any value-lift claim, so the messaging architecture must clear that tax before mentioning a named competitor.

## Diagnosis

Q2 loss-reason audit: 47% no-decision, 31% lost to a named competitor, 22% pricing. Above Dunford's 40% threshold for "no-decision-dominated pipeline."

## What the homepage now says

Old game / new game framing on the hero. Use-case epiphany above the fold. Setup-Follow-Through pitch architecture in the demo flow.

## Why we picked this position

[contradiction:no-decision-vs-named-competitor] picked Position A because the loss audit shows >40% no-decision losses, which is exactly the conditioning signal Dunford's framework names: pipeline shape decides allocation. We invest narrative work first, battle-card work second. Will re-evaluate if the next quarterly audit shows the mix has flipped.
